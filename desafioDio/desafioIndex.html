O que são Clients?

É um termo utilizado no meio do mundo da programação para se refir as maquinas do úsuario(cliente), pensando nisso, quando é criado um site ou aplicativo tem que ser pensando quais cógios serão executados na máquina do usuário, o que o usuário pode ter acesso em questão de código, o que deve ser carregado primeiro entre outras questões semelhantes.

O que são Servers?

É um termo utilizado no meio do mundo da programação para se referir a servidores que armazenam dados dos usuários que utilizam determinado aplicativo ou site. O usuário não deve ter acesso a todos os dados de um banco de dados dentro de um servidor e há código da progamação do servidor na qual o usuário não deve ter acesso a ele.

Como é uma estrutura básica de um HTML?

Um html é composto de tags, são marcações que determina como dertminado texto deve se comportar e a estruturação de cada elemento. As tags principais que todo html tem é o html,head e body.
